### A christmas gifts SPA website.

### Features:

- Animated modals
- Navigation menu for narrow screens
- Slider
- Category tabs switch
- Floating scroll up button

**Link to project:**
<a href="https://alexandermisyura-christmas-shop.netlify.app/">alexandermisyura-christmas-shop.netlify.app</a>

A single page application written in vanilla javascript with JSDoc type annotations. Client-side routing, interactive elements, smooth motions and animations. State management with Observer pattern. The website has been developed pixel-perfect based on [Figma layout](https://www.figma.com/design/zTB01BwWZVoXYK5atH3eZT/Cristmas-Shop)
